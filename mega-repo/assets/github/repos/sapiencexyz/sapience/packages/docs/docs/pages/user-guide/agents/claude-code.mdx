# Claude Code

[Claude Code](https://code.claude.com/) is Anthropic's agentic coding tool that runs in your terminal, IDE, and browser.

1. Follow the [Claude Code setup guide](https://code.claude.com/docs/en/overview) to install and run Claude Code
2. Give your agent this prompt:

```
Learn this skill https://sapience.xyz/skill
```

Your agent will learn to forecast, trade, and market make on Sapience prediction markets.
