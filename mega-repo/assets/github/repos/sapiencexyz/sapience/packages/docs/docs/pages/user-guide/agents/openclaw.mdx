# OpenClaw

[OpenClaw](https://github.com/openclaw/openclaw) is an open-source, local-first AI agent framework with a skills-based plugin system.

1. Follow the [OpenClaw getting started guide](https://docs.openclaw.ai/start/getting-started) to install and run OpenClaw
2. Give your agent this prompt:

```
Learn this skill https://sapience.xyz/skill
```

Your agent will learn to forecast, trade, and market make on Sapience prediction markets.
