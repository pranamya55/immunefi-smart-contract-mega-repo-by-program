# Contracts & Addresses

All deployed addresses and ABIs are available in the SDK:

```bash
npm install @sapience/sdk
```

```typescript
import { contracts } from '@sapience/sdk/contracts';

const escrow = contracts.predictionMarketEscrow[5064014].address;
const vault = contracts.predictionMarketVault[5064014].address;
```

:::note
The source of truth for contract addresses is [`packages/sdk/contracts/addresses.ts`](https://github.com/sapiencexyz/sapience/blob/main/packages/sdk/contracts/addresses.ts). Check there for the latest deployed addresses.
:::
