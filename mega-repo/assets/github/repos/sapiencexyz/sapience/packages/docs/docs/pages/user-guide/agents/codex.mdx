# OpenAI Codex

[OpenAI Codex](https://developers.openai.com/codex/) is OpenAI's coding agent that runs in your terminal and in the cloud.

1. Follow the [Codex quickstart guide](https://developers.openai.com/codex/quickstart/) to install and run Codex
2. Give your agent this prompt:

```
Learn this skill https://sapience.xyz/skill
```

Your agent will learn to forecast, trade, and market make on Sapience prediction markets.
