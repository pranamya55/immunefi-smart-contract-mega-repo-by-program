# Brand Assets

## Core Identity

- **Name:** Sapience
- **Tagline:** "Forecast the future with next-gen prediction markets"
- **Pillars:** Transparent · Permissionless · Open Source

## Logo & Avatar

The Sapience logo is a stylized profile icon with horizontal lines. Always use it on a dark background.

- **Orange/gold variant** — main brand and sports
- **Green variant** — developer account
- **Avatar** — used for app icon, social profiles, and favicons

Logo files are available on [GitHub](https://github.com/sapiencexyz/sapience/tree/main/packages/app/public).

## Colors

### Brand Neutrals

| Token           | Hex       | Usage                                     |
| --------------- | --------- | ----------------------------------------- |
| **Brand Black** | `#0B0B0A` | Primary background                        |
| **Brand White** | `#F7F5F3` | Primary text (warm white, not pure white) |
| **Foreground**  | `#E5D7C1` | Body text (warm tan)                      |
| **Background**  | `#151513` | App background                            |

### Brand Accents

| Token           | Hex       | Usage                                              |
| --------------- | --------- | -------------------------------------------------- |
| **Accent Gold** | `#F5D97E` | Primary accent — links, highlights, CTAs, emphasis |

| **Brand Red**   | `#E74C3C` | NO outcomes, destructive actions, down indicators  |

## Typography

| Family                  | Usage                |
| ----------------------- | -------------------- |
| **Avenir Next**         | Headings and display |
| **Avenir Next Rounded** | Body text            |
| **IBM Plex Mono**       | Monospace / code     |

### Text Styles

| Style        | Usage                      | Properties                                       |
| ------------ | -------------------------- | ------------------------------------------------ |
| **Eyebrow**  | Section labels, categories | Uppercase, medium weight, wide tracking (0.12em) |
| **Headline** | Page and section titles    | Avenir Next, xl–3xl responsive                   |
